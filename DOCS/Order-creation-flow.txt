1. 
Obtener Configuraciones Base
•  Llamar a: /api/v1/company/settings

•  Descripción: Este endpoint permite obtener los valores de configuraciones base necesarios para realizar transacciones en la plataforma. En el caso del POS, permite:

•  Indicar si el número de factura o de pago se autogeneran o si el usuario debe ingresarlos manualmente.

•  Establecer los días de diferencia entre la fecha de creación de la factura y su fecha de vencimiento.

•  Determinar si se activa el formato PDF para POS o se usa el formato normal.

2. 
Obtener el Número de Documento para la Factura
•  Llamar a: /api/v1/next-number

•  Descripción: Este endpoint proporciona el siguiente número disponible para la factura y el prefijo utilizado.

3. 
Crear la Factura
•  Llamar a: /api/v1/invoices

•  Descripción: Este endpoint crea la factura y retorna el ID de la misma. Este ID puede ser utilizado para crear el pago inmediatamente o para guardar la factura.

4. 
Obtener Métodos de Pago para POS
•  Llamar a: /api/v1/payments/multiple/get-payment-methods

•  Descripción: Este endpoint permite obtener los métodos de pago exclusivos para POS, junto con las denominaciones de billetes asociadas y las tarifas (fees).

5. 
Obtener el Número de Documento para el Pago
•  Llamar a: /api/v1/next-number

•  Descripción: Este endpoint proporciona el siguiente número disponible para el pago y el prefijo utilizado.

6. 
Consultar la Factura
•  Llamar a: /invoices/{id}

•  Descripción: Este endpoint permite obtener la información de la factura ya creada según el ID proporcionado.

7. 
Crear el Pago de la Factura
•  Llamar a: /api/v1/payments/multiple/create

•  Descripción: Este endpoint permite generar un pago utilizando múltiples métodos de pago y retorna el ID del pago realizado.

8. 
Consultar el Pago
•  Llamar a: /payments/{id}

•  Descripción: Este endpoint permite ver u obtener la información del pago realizado enviando el ID del mismo.
